# Demo Release Folder

This folder is for local demo release packaging only.

The Windows demo installer ZIP should be generated locally as:

```text
release/HospitalFLSystem_Demo_Setup.zip
```

The ZIP is intentionally ignored by Git because it is a large generated binary. Upload it to GitHub Releases instead of committing it to the normal source repository.

Recommended GitHub Release asset name:

```text
HospitalFLSystem_Demo_Setup.zip
```
